# 🍕 La Gastronomie Pizza – Digital Menu

This project is a **mobile-friendly digital restaurant menu** designed to be accessed through a **QR code**. Customers can scan the QR code and view the menu directly on their phone.

The menu includes **images, prices, item descriptions, and language switching (French / English)**.
